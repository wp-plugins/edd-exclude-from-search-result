This Plugin Automatically Exclude Downloads Search Results In Easy Digital Downloads.

== Description ==

This Plugin Automatically Exclude Downloads Search Results In Easy Digital Downloads. Just Activate this Plugin and enjoy no settings needed.

The EDD Exclude From Search Result plugin ONLY works with Easy Digital Downloads 1.8.3 or later plugins is installed.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'EDD Exclude From Search Result'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard
5. That's it!

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select 'edd-exclude-from-search-result.zip' from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download 'edd-exclude.zip'
2. Extract the 'edd-exclude-from-search-result.zip' directory to your computer
3. Upload the 'edd-exclude-from-search-result' directory to the '/wp-content/plugins/' directory
4. Activate the plugin in the Plugin dashboard
